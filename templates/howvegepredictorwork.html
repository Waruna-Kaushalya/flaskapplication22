Import Data and Train the Model

Import the data required to train the model into the application in CSV format. (Enter data for at least ten years.)



A format of the CSV type data to be entered.


How to create Create a CSV file

First, select the "choose file" button and then the CSV file and click the "upload" button.







It will take some time to upload the file.



Then click on the "Train model" button. It takes some time for the model to train.


After the model trained you can go to "Single prediction" and predict values.


Single prediction

Enter the data needed to predict here. Include the average value of two months' weather, as it usually takes about two months for an up-country vegetable to harvest.



Then click on the "Predict the value" button to prediction. The result related to the prediction is displayed as follows.
